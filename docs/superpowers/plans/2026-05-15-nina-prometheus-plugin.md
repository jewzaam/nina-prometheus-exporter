# NINA Prometheus Exporter Plugin Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a NINA plugin that exposes a Prometheus scrape endpoint on `http://<bind>:<port>/metrics`, replacing the log-tailing `log-exporter-nina.yaml` + `log-exporter-autofocus.yaml` and adding the Section 5 additional metrics from the design.

**Architecture:** A single .NET 8 NINA plugin (`NINA.Plugin.PrometheusExporter`). One `PrometheusExporterPlugin` class wires `[ImportingConstructor]` MEF injection of 14 NINA mediators into per-family collector classes (one per `Stream/*.cs` file). Each collector implements a `*Consumer` interface (where applicable) and subscribes to its mediator's events. A `PrometheusServer` wraps `prometheus-net`'s `MetricServer`. Settings are persisted via `IPluginOptionsAccessor`; changes apply without NINA restart by raising `PropertyChanged` to the plugin's handler.

- .NET 8 (`net8.0`), C# 12
- `NINA.Plugin` NuGet (latest at first build; later pinned to oldest 3.x where every used API compiles — see spec §14)

# NINA Prometheus Exporter

NINA 3.x plugin that exposes a Prometheus scrape endpoint. Replaces log-tailing for NINA-sourced metrics.

## Install

Either:
1. Install via NINA's in-app plugin manager (once published), or
2. Download the latest `.zip` from [Releases](https://github.com/jewzaam/nina-prometheus-exporter/releases) and extract its contents into `%LOCALAPPDATA%\NINA\Plugins\Prometheus Exporter\`.

## Configure

In NINA → Options → Plugins → Prometheus Exporter:
- **Port** (default `9876`)
- **Bind address** (default `0.0.0.0`; loopback `127.0.0.1` keeps it local-only)
- **Sequence poll interval (seconds)** (default `1`)
- **Autofocus timeout (minutes)** (default `10`; `0` disables)

The exporter is always on while the plugin is loaded. Uninstall the plugin to stop it.

## Security

The default `0.0.0.0` bind exposes the metrics endpoint on every network interface. There is no authentication. If your NINA host is not on a trusted LAN, change the bind to `127.0.0.1` or use a host firewall to restrict access.

// Generated By: Claude Code (Claude Opus 4.7)
using NINA.Core.Utility;
using NINA.Plugin;
using NINA.Plugin.Interfaces;
using NINA.Profile.Interfaces;
using System.ComponentModel;
using System.Runtime.CompilerServices;

namespace NINA.Plugin.PrometheusExporter {

    public sealed class PrometheusExporterOptions : BaseINPC {
        private readonly IPluginOptionsAccessor _accessor;

        public PrometheusExporterOptions(IProfileService profileService) {
            // Use the plugin's own AssemblyGuid for the options namespace.
            _accessor = new PluginOptionsAccessor(profileService, System.Guid.Parse(
                typeof(PrometheusExporterOptions).Assembly
                    .GetCustomAttributes(typeof(System.Runtime.InteropServices.GuidAttribute), false)
                    .Cast<System.Runtime.InteropServices.GuidAttribute>()
                    .First().Value));
        }

        public int Port {
            get => _accessor.GetValueInt32(nameof(Port), 9876);
            set { _accessor.SetValueInt32(nameof(Port), Clamp(value, 1, 65535)); RaisePropertyChanged(); }
        }

        public string BindAddress {
            get => _accessor.GetValueString(nameof(BindAddress), "0.0.0.0");
            set { _accessor.SetValueString(nameof(BindAddress), value ?? "0.0.0.0"); RaisePropertyChanged(); }
        }

        public int SequencePollIntervalSeconds {
            get => _accessor.GetValueInt32(nameof(SequencePollIntervalSeconds), 1);
            set { _accessor.SetValueInt32(nameof(SequencePollIntervalSeconds), Clamp(value, 1, 60)); RaisePropertyChanged(); }
        }

        public int AfTimeoutMinutes {
            get => _accessor.GetValueInt32(nameof(AfTimeoutMinutes), 10);
            set { _accessor.SetValueInt32(nameof(AfTimeoutMinutes), Clamp(value, 0, 120)); RaisePropertyChanged(); }
        }

        private static int Clamp(int v, int lo, int hi) => v < lo ? lo : (v > hi ? hi : v);
    }
}
```

If `BaseINPC` is not the right base class in NINA 3.x (check `NINA.Core.Utility.BaseINPC`), substitute the equivalent NINA observable base or implement `INotifyPropertyChanged` manually. The `IPluginOptionsAccessor` API surface above (`GetValueInt32`, `SetValueInt32`, `GetValueString`, `SetValueString`) is what NINA exposes; if a different method is required for your `NINA.Plugin` version, adapt the calls.

using NINA.Core.Utility;
using Prometheus;
using System;

namespace NINA.Plugin.PrometheusExporter {

    internal sealed class PrometheusServer {
        private MetricServer? _server;
        public string Status { get; private set; } = "stopped";

        public void Start(string bindAddress, int port) {
            try {
                _server = new MetricServer(hostname: bindAddress, port: port);
                _server.Start();
                Status = $"running on {bindAddress}:{port}";
                Logger.Info($"PrometheusServer: {Status}");
            } catch (Exception ex) {
                Status = $"failed: {ex.Message}";
                Logger.Error($"PrometheusServer start failed: {ex}");
            }
        }

        public void Stop() {
            try {
                _server?.Stop();
                _server?.Dispose();
            } catch (Exception ex) {
                Logger.Error($"PrometheusServer stop failed: {ex}");
            } finally {
                _server = null;
                Status = "stopped";
            }
        }

        public void Restart(string bindAddress, int port) {
            Stop();
            Start(bindAddress, port);
        }
    }
}

        private readonly Gauge _gauge;

        private readonly ICameraMediator _camera;
        private readonly ITelescopeMediator _telescope;
        private readonly IFocuserMediator _focuser;
        private readonly IFilterWheelMediator _filterWheel;

        private readonly IFlatDeviceMediator _flat;
        private readonly ISafetyMonitorMediator _safety;
        private readonly IWeatherDataMediator _weather;
        private readonly ISwitchMediator _switch;

        public EquipmentMetrics(
            MetricFactory factory,
            ICameraMediator camera,
            ITelescopeMediator telescope,
            IFocuserMediator focuser,
            IFilterWheelMediator filterWheel,
            IGuiderMediator guider,
            IDomeMediator dome,
            IRotatorMediator rotator,
            IFlatDeviceMediator flat,
            ISafetyMonitorMediator safety,
            IWeatherDataMediator weather,
            ISwitchMediator @switch) {
            _factory = factory;
            _camera = camera; _telescope = telescope; _focuser = focuser;
            _filterWheel = filterWheel; _guider = guider; _dome = dome;
            _rotator = rotator; _flat = flat; _safety = safety;
            _weather = weather; _switch = @switch;

            _gauge = Metrics.CreateGauge(
                "nina_equipment",
                "1 if the equipment of the given type is connected, 0 otherwise",
                new GaugeConfiguration { LabelNames = _factory.LabelNames("type") });
        }

        public void Subscribe() {
            Wire(_camera, "camera");
            Wire(_telescope, "telescope");
            Wire(_focuser, "focuser");
            Wire(_filterWheel, "filterwheel");
            Wire(_guider, "guider");
            Wire(_dome, "dome");
            Wire(_rotator, "rotator");
            Wire(_flat, "flat_device");
            Wire(_safety, "safety_monitor");
            Wire(_weather, "weather");
            Wire(_switch, "switch");
        }

        private void Wire<THandler, TConsumer, TInfo>(
            IDeviceMediator<THandler, TConsumer, TInfo> mediator,
            string type)
            where THandler : NINA.Core.Interfaces.IDeviceVM<TInfo>
            where TConsumer : NINA.Equipment.Interfaces.IDeviceConsumer<TInfo> {

            mediator.Connected += (sender, args) => {
                _gauge.WithLabels(_factory.LabelValues(type)).Set(1);
                return Task.CompletedTask;
            };
            mediator.Disconnected += (sender, args) => {
                _gauge.WithLabels(_factory.LabelValues(type)).Set(0);
                return Task.CompletedTask;
            };
        }

        public void Dispose() {
            // The handlers above are anonymous closures and not retained for removal here.
            // Disposed at NINA shutdown when the mediators themselves go away.
            GC.SuppressFinalize(this);
        }
    }
}
```

The `Wire<>` helper uses the generic `IDeviceMediator<H, C, I>` base interface that all the per-device mediators implement (confirmed at `~/source/nina/NINA.Equipment/Interfaces/Mediator/IDeviceMediator.cs:23`). The `Connected` / `Disconnected` events are inherited from that base. `IWeatherDataMediator`, `ISwitchMediator` etc. all extend it.

Anonymous closures intentionally not retained — `Dispose()` is a no-op for this collector since plugin teardown means the AppDomain (and the mediators) goes away. If we add explicit unsubscribe later, store the delegates as fields and remove them in `Dispose()`.

// Generated By: Claude Code (Claude Opus 4.7)
using NINA.Core.Enum;
using NINA.Equipment.Equipment.MyTelescope;
using NINA.Equipment.Interfaces.Mediator;
using Prometheus;
using System;
using System.Threading.Tasks;

namespace NINA.Plugin.PrometheusExporter.Stream {

    internal sealed class MountMetrics : IMetricCollector, ITelescopeConsumer {
        private readonly MetricFactory _factory;
        private readonly ITelescopeMediator _mediator;

        private readonly Gauge _altitude;
        private readonly Gauge _azimuth;
        private readonly Gauge _ra;
        private readonly Gauge _dec;
        private readonly Gauge _sideOfPier;        // 0=Unknown, 1=East, 2=West
        private readonly Gauge _tracking;          // 0/1
        private readonly Gauge _slewing;           // 0/1
        private readonly Gauge _parked;            // 0/1
        private readonly Gauge _meridianFlip;      // 0/1, 1 during flip
        private readonly Counter _sopUnknown;      // edge counter into Unknown

        private PierSide _lastSop = PierSide.pierUnknown;

        public MountMetrics(MetricFactory factory, ITelescopeMediator mediator) {
            _factory = factory;
            _mediator = mediator;

            var ln = _factory.LabelNames();
            _altitude     = Metrics.CreateGauge("nina_mount_altitude_degrees",   "Mount altitude in degrees",         new GaugeConfiguration { LabelNames = ln });
            _azimuth      = Metrics.CreateGauge("nina_mount_azimuth_degrees",    "Mount azimuth in degrees",          new GaugeConfiguration { LabelNames = ln });
            _ra           = Metrics.CreateGauge("nina_mount_ra_hours",           "Mount RA in hours",                 new GaugeConfiguration { LabelNames = ln });
            _dec          = Metrics.CreateGauge("nina_mount_dec_degrees",        "Mount Dec in degrees",              new GaugeConfiguration { LabelNames = ln });
            _sideOfPier   = Metrics.CreateGauge("nina_mount_side_of_pier",       "Mount side of pier (0=Unknown, 1=East, 2=West)", new GaugeConfiguration { LabelNames = ln });
            _tracking     = Metrics.CreateGauge("nina_mount_tracking",           "1 if tracking enabled, else 0",     new GaugeConfiguration { LabelNames = ln });
            _slewing      = Metrics.CreateGauge("nina_mount_slewing",            "1 if currently slewing, else 0",    new GaugeConfiguration { LabelNames = ln });
            _parked       = Metrics.CreateGauge("nina_mount_parked",             "1 if parked, else 0",               new GaugeConfiguration { LabelNames = ln });
            _meridianFlip = Metrics.CreateGauge("nina_mount_meridian_flip",      "1 between BeforeMeridianFlip and AfterMeridianFlip events, else 0", new GaugeConfiguration { LabelNames = ln });

            _sopUnknown = Metrics.CreateCounter(
                "nina_mount_side_of_pier_unknown_total",
                "Edge-triggered count of side-of-pier transitions into Unknown",
                new CounterConfiguration { LabelNames = ln });
        }

        public void Subscribe() {
            _mediator.RegisterConsumer(this);
            _mediator.BeforeMeridianFlip += OnBefore;
            _mediator.AfterMeridianFlip  += OnAfter;
        }

        public void UpdateDeviceInfo(TelescopeInfo info) {
            if (info == null || !info.Connected) return;

            var lv = _factory.LabelValues();
            if (!double.IsNaN(info.Altitude)) _altitude.WithLabels(lv).Set(info.Altitude);
            if (!double.IsNaN(info.Azimuth))  _azimuth.WithLabels(lv).Set(info.Azimuth);
            if (info.Coordinates != null) {
                _ra.WithLabels(lv).Set(info.Coordinates.RA);
                _dec.WithLabels(lv).Set(info.Coordinates.Dec);
            }

            _tracking.WithLabels(lv).Set(info.TrackingEnabled ? 1 : 0);
            _slewing.WithLabels(lv).Set(info.Slewing ? 1 : 0);
            _parked.WithLabels(lv).Set(info.AtPark ? 1 : 0);

            var sop = info.SideOfPier;
            _sideOfPier.WithLabels(lv).Set(sop switch {
                PierSide.pierEast    => 1,
                PierSide.pierWest    => 2,
                _                    => 0
            });
            if (sop == PierSide.pierUnknown && _lastSop != PierSide.pierUnknown)
                _sopUnknown.WithLabels(lv).Inc();
            _lastSop = sop;
        }

        private Task OnBefore(object sender, BeforeMeridianFlipEventArgs args) {
            _meridianFlip.WithLabels(_factory.LabelValues()).Set(1);
            return Task.CompletedTask;
        }

        private Task OnAfter(object sender, AfterMeridianFlipEventArgs args) {
            _meridianFlip.WithLabels(_factory.LabelValues()).Set(0);
            return Task.CompletedTask;
        }

        public void Dispose() {
            _mediator.BeforeMeridianFlip -= OnBefore;
            _mediator.AfterMeridianFlip  -= OnAfter;
            _mediator.RemoveConsumer(this);
            GC.SuppressFinalize(this);
        }
    }
}

using NINA.Core.Utility;
using NINA.Sequencer.Container;
using NINA.Sequencer.Interfaces.Mediator;
using Prometheus;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Threading;

namespace NINA.Plugin.PrometheusExporter.Stream {

    internal sealed class SequenceMetrics : IMetricCollector {
        private readonly MetricFactory _factory;
        private readonly ISequenceMediator _mediator;
        private readonly PrometheusExporterOptions _options;

        private readonly Gauge _status;
        private readonly Counter _started;
        private readonly Counter _completed;

        private readonly HashSet<(string Category, string Item)> _running = new();
        private readonly object _diffLock = new();
        private Timer? _timer;

        public SequenceMetrics(MetricFactory factory, ISequenceMediator mediator, PrometheusExporterOptions options) {
            _factory = factory;
            _mediator = mediator;
            _options = options;

            var ln = _factory.LabelNames("category", "item");
            _status    = Metrics.CreateGauge("nina_status",                       "1 while sequence item is running, else 0", new GaugeConfiguration { LabelNames = ln });
            _started   = Metrics.CreateCounter("nina_status_count_started_total", "Count of sequence-item runs started",      new CounterConfiguration { LabelNames = ln });
            _completed = Metrics.CreateCounter("nina_status_count_completed_total","Count of sequence-item runs completed",   new CounterConfiguration { LabelNames = ln });
        }

        public void Subscribe() => StartPolling();

        public void StartPolling() {
            var ms = _options.SequencePollIntervalSeconds * 1000;
            _timer = new Timer(_ => Poll(), null, ms, ms);
        }

        public void RestartTimer(int newSeconds) {
            _timer?.Dispose();
            var ms = newSeconds * 1000;
            _timer = new Timer(_ => Poll(), null, ms, ms);
        }

        public void StopPolling() {
            _timer?.Dispose();
            _timer = null;
        }

        internal void Poll() {
            HashSet<(string, string)> current;
            try {
                current = SnapshotRunning();
            } catch (Exception ex) {
                Logger.Debug($"SequenceMetrics: GetAdvancedSequencerCurrentRunningItems threw — mediator not ready? {ex.Message}");
                return;
            }

            lock (_diffLock) {
                foreach (var added in current.Except(_running)) {
                    var lv = _factory.LabelValues(added.Item1, added.Item2);
                    _started.WithLabels(lv).Inc();
                    _status.WithLabels(lv).Set(1);
                }
                foreach (var removed in _running.Except(current)) {
                    var lv = _factory.LabelValues(removed.Item1, removed.Item2);
                    _completed.WithLabels(lv).Inc();
                    _status.WithLabels(lv).Set(0);
                }
                _running.Clear();
                foreach (var c in current) _running.Add(c);
            }
        }

        private HashSet<(string, string)> SnapshotRunning() {
            var set = new HashSet<(string, string)>();
            var groups = _mediator.GetAdvancedSequencerCurrentRunningItems();
            if (groups == null) return set;
            foreach (var group in groups) {
                if (group == null) continue;
                foreach (var item in group) {
                    if (item == null) continue;
                    var category = (item.Parent as ISequenceContainer)?.Name ?? "unknown";
                    var name = item.Name ?? item.GetType().Name;
                    set.Add((category, name));
                }
            }
            return set;
        }

        public void Dispose() {
            StopPolling();
            GC.SuppressFinalize(this);
        }
    }
}

//   $.RSquares.Hyperbolic       → double (may be missing)
//   $.RSquares.LeftTrend        → double (may be missing)
//   $.RSquares.RightTrend       → double (may be missing)
//
// Update these paths if the captured golden file diverges.

using System.Collections.Generic;
using System.IO;
using System.Text.Json;

namespace NINA.Plugin.PrometheusExporter.Stream {

    internal sealed class AutoFocusReport {
        public string Filter { get; set; } = "unknown";
        public string Method { get; set; } = "unknown";
        public string Fitting { get; set; } = "unknown";
        public int BacklashIn { get; set; }
        public int BacklashOut { get; set; }
        public Dictionary<string, double> RSquares { get; } = new();
    }

    internal static class AutoFocusReportParser {

        public static AutoFocusReport ParseFile(string path) {
            using var stream = File.OpenRead(path);
            using var doc = JsonDocument.Parse(stream);
            return Parse(doc.RootElement);
        }

        public static AutoFocusReport Parse(JsonElement root) {
            var report = new AutoFocusReport {
                Filter = GetString(root, "Filter", "unknown"),
                Method = GetString(root, "Method", "unknown"),
                Fitting = GetString(root, "Fitting", "unknown")
            };
            if (root.TryGetProperty("BacklashCompensation", out var bc)) {
                report.BacklashIn = GetInt(bc, "BacklashIN", 0);
                report.BacklashOut = GetInt(bc, "BacklashOUT", 0);
            }
            if (root.TryGetProperty("RSquares", out var rs)) {
                foreach (var prop in rs.EnumerateObject()) {
                    if (prop.Value.ValueKind == JsonValueKind.Number)
                        report.RSquares[prop.Name] = prop.Value.GetDouble();
                }
            }
            return report;
        }

        private static string GetString(JsonElement el, string name, string fallback) =>
            el.TryGetProperty(name, out var p) && p.ValueKind == JsonValueKind.String ? p.GetString()! : fallback;

        private static int GetInt(JsonElement el, string name, int fallback) =>
            el.TryGetProperty(name, out var p) && p.TryGetInt32(out var v) ? v : fallback;
    }
}

using NINA.Equipment.Equipment.MyFocuser;
using NINA.Equipment.Interfaces.Mediator;
using OxyPlot;
using Prometheus;

using System.IO;
using System.Threading;

namespace NINA.Plugin.PrometheusExporter.Stream {

    internal sealed class AutoFocusMetrics : IMetricCollector, IFocuserConsumer {
        private readonly MetricFactory _factory;
        private readonly IFocuserMediator _mediator;
        private readonly PrometheusExporterOptions _options;

        private readonly Gauge _running;
        private readonly Counter _successes;
        private readonly Counter _failures;
        private readonly Gauge _rsquares;

        // State machine
        private enum AfState { Idle, Running }
        private AfState _state = AfState.Idle;
        private readonly object _stateLock = new();
        private Timer? _timeoutTimer;

        // JSON watcher
        private FileSystemWatcher? _watcher;
        private readonly Timer _debounce;
        private string? _pendingPath;

        public AutoFocusMetrics(MetricFactory factory, IFocuserMediator mediator, PrometheusExporterOptions options) {
            _factory = factory;
            _mediator = mediator;
            _options = options;

            var ln = _factory.LabelNames();
            _running   = Metrics.CreateGauge("nina_autofocus_running",         "1 while an autofocus run is in progress, else 0", new GaugeConfiguration { LabelNames = ln });
            _successes = Metrics.CreateCounter("nina_autofocus_success_total", "Count of successful autofocus runs",              new CounterConfiguration { LabelNames = _factory.LabelNames("filter") });
            _failures  = Metrics.CreateCounter("nina_autofocus_failure_total", "Count of failed autofocus runs",                  new CounterConfiguration { LabelNames = _factory.LabelNames("reason") });
            _rsquares  = Metrics.CreateGauge("nina_autofocus_rsquares",        "R-squared of fit type for last autofocus report", new GaugeConfiguration {
                LabelNames = _factory.LabelNames("filter", "method", "fitting", "backlash_in", "backlash_out", "type")
            });

            _debounce = new Timer(_ => ParsePending(), null, Timeout.Infinite, Timeout.Infinite);
        }

        public void Subscribe() {
            _mediator.RegisterConsumer(this);
            StartWatcher();
        }

        // === IFocuserConsumer state-machine callbacks ===

        public void AutoFocusRunStarting() {
            lock (_stateLock) {
                if (_state == AfState.Running) {
                    Logger.Debug("AutoFocusMetrics: duplicate AutoFocusRunStarting ignored");
                    return;
                }
                _state = AfState.Running;
                _running.WithLabels(_factory.LabelValues()).Set(1);
                if (_options.AfTimeoutMinutes > 0) {
                    var ms = (long)TimeSpan.FromMinutes(_options.AfTimeoutMinutes).TotalMilliseconds;
                    _timeoutTimer?.Dispose();
                    _timeoutTimer = new Timer(_ => OnTimeout(), null, ms, Timeout.Infinite);
                }
            }
        }

        public void UpdateEndAutoFocusRun(AutoFocusInfo info) {
            lock (_stateLock) {
                if (_state == AfState.Idle) {
                    Logger.Debug("AutoFocusMetrics: UpdateEndAutoFocusRun while Idle — ignored");
                    return;
                }
                _state = AfState.Idle;
                _running.WithLabels(_factory.LabelValues()).Set(0);
                _timeoutTimer?.Dispose();
                _timeoutTimer = null;
                _successes.WithLabels(_factory.LabelValues(info?.Filter ?? "unknown")).Inc();
            }
        }

        private void OnTimeout() {
            lock (_stateLock) {
                if (_state == AfState.Idle) return;
                _state = AfState.Idle;
                _running.WithLabels(_factory.LabelValues()).Set(0);
                _failures.WithLabels(_factory.LabelValues("timeout")).Inc();
                Logger.Info($"AutoFocusMetrics: timeout fired after {_options.AfTimeoutMinutes} min");
            }
        }

        // No-op consumer callbacks owned by FocuserMetrics:
        public void UpdateDeviceInfo(FocuserInfo info) { }
        public void UpdateUserFocused(FocuserInfo info) { }
        public void NewAutoFocusPoint(DataPoint dataPoint) { }

        // === JSON watcher ===

        private void StartWatcher() {
            var dir = Path.Combine(Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData), "NINA", "AutoFocus");
            if (!Directory.Exists(dir)) Directory.CreateDirectory(dir);

            _watcher = new FileSystemWatcher(dir, "*.json") {
                NotifyFilter = NotifyFilters.LastWrite | NotifyFilters.CreationTime | NotifyFilters.FileName,
                IncludeSubdirectories = false,
                EnableRaisingEvents = true
            };
            _watcher.Created += OnFileEvent;
            _watcher.Changed += OnFileEvent;
        }

        private void OnFileEvent(object sender, FileSystemEventArgs e) {
            _pendingPath = e.FullPath;
            _debounce.Change(200, Timeout.Infinite);
        }

        private void ParsePending() {
            var path = _pendingPath;
            if (path == null || !File.Exists(path)) return;
            try {
                var report = AutoFocusReportParser.ParseFile(path);
                foreach (var (fitType, value) in report.RSquares) {
                    _rsquares.WithLabels(_factory.LabelValues(
                        report.Filter,
                        report.Method,
                        report.Fitting,
                        report.BacklashIn.ToString(),
                        report.BacklashOut.ToString(),
                        fitType
                    )).Set(value);
                }
            } catch (Exception ex) {
                Logger.Debug($"AutoFocusMetrics: parse failed for {path}: {ex.Message}");
            }
        }

        public void Dispose() {
            _watcher?.Dispose();
            _debounce.Dispose();
            _timeoutTimer?.Dispose();
            _mediator.RemoveConsumer(this);
            GC.SuppressFinalize(this);
        }
    }
}

using System;

namespace NINA.Plugin.PrometheusExporter.Stream {

    internal sealed class AutoFocusStateMachine {
        private readonly Action<bool> _onRunningChanged;
        private readonly Action<string> _onSuccess;            // arg: filter
        private readonly Action<string> _onFailure;            // arg: reason
        private readonly object _lock = new();
        private bool _running;

        public AutoFocusStateMachine(
            Action<bool> onRunningChanged,
            Action<string> onSuccess,
            Action<string> onFailure) {
            _onRunningChanged = onRunningChanged;
            _onSuccess = onSuccess;
            _onFailure = onFailure;
        }

        public bool IsRunning { get { lock (_lock) return _running; } }

        public void Start() {
            lock (_lock) {
                if (_running) return;       // idempotent
                _running = true;
            }
            _onRunningChanged(true);
        }

        public void End(string filter) {
            lock (_lock) {
                if (!_running) return;      // idempotent
                _running = false;
            }
            _onRunningChanged(false);
            _onSuccess(filter);
        }

        public void Timeout() {
            lock (_lock) {
                if (!_running) return;      // race: end fired first
                _running = false;
            }
            _onRunningChanged(false);
            _onFailure("timeout");
        }
    }
}

        public void UpdateDeviceInfo(WeatherDataInfo info) {
            if (info == null || !info.Connected) return;
            var lv = _factory.LabelValues();
            void S(Gauge g, double v) { if (!double.IsNaN(v)) g.WithLabels(lv).Set(v); }
            S(_tempC, info.Temperature);
            S(_humidity, info.Humidity);
            S(_pressure, info.Pressure);
            S(_cloud, info.CloudCover);
            S(_windSpeed, info.WindSpeed);
            S(_windDir, info.WindDirection);
            S(_dewPoint, info.DewPoint);
            S(_skyTemp, info.SkyTemperature);
            S(_skyBrightness, info.SkyBrightness);
            S(_skyQuality, info.SkyQuality);
            S(_rainRate, info.RainRate);
        }

        public void Dispose() {
            _mediator.RemoveConsumer(this);
            GC.SuppressFinalize(this);
        }

}

// In ctor (after _options instantiation):

private void OnOptionsChanged(object? sender, System.ComponentModel.PropertyChangedEventArgs e) {
    switch (e.PropertyName) {
        case nameof(PrometheusExporterOptions.Port):
        case nameof(PrometheusExporterOptions.BindAddress):
            _server.Restart(_options.BindAddress, _options.Port);
            break;
        case nameof(PrometheusExporterOptions.SequencePollIntervalSeconds):
            _sequence.RestartTimer(_options.SequencePollIntervalSeconds);
            break;
        case nameof(PrometheusExporterOptions.AfTimeoutMinutes):
            // Picked up on next AF run; nothing to restart now.
            break;
    }
}

// In Teardown(), before stopping the server:
_options.PropertyChanged -= OnOptionsChanged;
